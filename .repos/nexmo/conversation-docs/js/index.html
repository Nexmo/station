Nexmo Stitch SDK for JavaScript

The Conversation SDK is intended to provide a ready solution for developers who want to integrate chat, voice and video into their apps.

Installation

NPM

A dependency manager for Node packages. You can install the Conversation SDK with the following command:

$ npm install nexmo-stitch

SDK Setup

Include the conversationClient.js script in your web page

<script src="node_modules/nexmo-stitch/dist/conversationClient.js"></script>

Testing

We use third party tools for testing the framework:

  • Karma: Karma is our test runner
  • Mocha: Mocha is a feature-rich JavaScript test framework running on Node.js and in the browser, making asynchronous testing simple and fun.
  • Chai: Chai is a BDD / TDD assertion library for node and the browser that can be delightfully paired with any javascript testing framework.
